Google Search Clone - CS50 Web Track Project

A frontend-only replica of Google's search interface built as part of the CS50 Web track. This project recreates the core Google search experience with three main pages: the main search page, an advanced search interface, and a Google Images search portal.

Features:

Google Search homepage with search bar and voice/image search buttons
Advanced Search page with multiple filter options (exact phrase, included/excluded words, number range filters)
Google Images search interface
Responsive design with HTML/CSS styling
Forms that redirect to actual Google search results
Technologies:

HTML5
CSS3
Semantic HTML markup
Project Structure:

index.html - Main Google Search page
Advance.html - Advanced search filters
image.html - Google Images search
adv.css - Styling for advanced search
Supporting images in /image directory
