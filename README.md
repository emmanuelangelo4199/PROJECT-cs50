# Google Search Clone - CS50 Web Track Project

A frontend-only replica of Google's search interface built as part of the CS50 Web track. This project recreates the core Google search experience with multiple search interfaces.

## 📋 Project Overview

This project demonstrates fundamental web development skills including HTML structure, CSS styling, form handling, and responsive design. It consists of three main pages that mimic Google's search functionality.

## 🚀 Features

- **Google Search Homepage** - Clean search interface with Google's iconic design
  - Search bar with placeholder text
  - Voice search button (SVG icon)
  - Image search button (SVG icon)
  - "Google Search" and "I'm Feeling Lucky" buttons

- **Advanced Search Page** - Advanced filtering capabilities
  - Search for all these words
  - Search for exact phrase
  - Search for any of these words
  - Search for none of these words
  - Number range filter
  - Direct integration with Google's advanced search

- **Google Images Search** - Dedicated image search interface
  - Similar layout to main search page
  - Search form integrated with Google Images

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and form structure
- **CSS3** - Styling, flexbox layout, and hover effects
- **SVG Icons** - Voice and image search icons

## 🎯 How to Use

1. Open `index.html` in your web browser to access the main search page
2. Navigate to "Google Image" or "Google Advance" using the links at the top
3. Enter search queries and click "Google Search" to redirect to actual Google results
4. Experiment with the advanced search filters to refine your search criteria
